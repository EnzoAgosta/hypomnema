# Tutorial

Welcome to the Hypomnema tutorial. This guide will walk you through everything you need to know to work with TMX files in Python.

## What You'll Learn

- How to load and save TMX files
- How to create TMX documents from scratch
- How to work with translation units and variants
- How to handle inline markup (formatting tags, placeholders)
- How to process large files efficiently with streaming
- How to configure error handling for your use case

## Prerequisites

- Python 3.13 or later
- Basic familiarity with Python (dataclasses, type hints)

## Tutorial Roadmap

1. **Installation** — Get hypomnema installed and verify it works
2. **Your First TMX** — Load a file and explore its structure
3. **Creating TMX Files** — Build a TMX document from scratch
4. **Working with Variants** — Handle language pairs and segment content
5. **Inline Markup** — Work with formatting tags, placeholders, and highlights
6. **Saving Files** — Write TMX to disk with encoding options
7. **Streaming Large Files** — Process huge files without running out of memory
8. **Error Handling** — Configure how the library handles malformed data
9. **Backends** — Choose between stdlib and lxml backends

Ready? Let's start with [Installation](/en/tutorial/01-installation).
