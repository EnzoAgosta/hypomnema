# Cookbook

Practical recipes for common TMX processing tasks. Each recipe is self-contained and solves a specific problem.

## Available Recipes

### Data Conversion
- [CSV to TMX](/en/cookbook/csv-to-tmx) — Convert spreadsheets to translation memory format

### Analysis & Extraction
- [TMX Statistics](/en/cookbook/tmx-statistics) — Extract metrics from TMX files
- [Extract Plain Text](/en/cookbook/extract-plain-text) — Strip markup, get raw text

### File Operations
- [Merge TMX Files](/en/cookbook/merge-tmx-files) — Combine multiple translation memories
- [Split by Language](/en/cookbook/split-tmx-by-language) — Extract specific language pairs
- [Filter by Date](/en/cookbook/filter-by-date) — Keep only recent entries

### Validation & Conversion
- [Validate TMX](/en/cookbook/validate-tmx) — Check for malformed data
- [Convert Between Tools](/en/cookbook/convert-between-tools) — Handle tool-specific metadata
